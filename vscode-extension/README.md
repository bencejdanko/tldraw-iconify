# tldraw-iconify

[![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)](https://github.com/bencejdanko/tldraw-iconify)

The best way to edit .tldr files directly in VS Code. Featuring the full [tldraw](https://tldraw.com) experience with local-first persistence and Iconify integration.

![tldraw VS Code Extension Demo](./image.png)

## Features

- **Full tldraw Experience**: Use all the tools you love—shapes, arrows, handwriting, and more.
- **Native .tldr Support**: Open and save files with the .tldr extension natively.
- **Iconify Search**: Access 200,000+ icons directly within the whiteboard.
- **Privacy First**: Everything stays on your local machine. No cloud, no tracking.
- **Works Offline**: Keep drawing even without an internet connection.

## Usage

1. Create a new file with the .tldr extension (e.g., diagram.tldr).
2. The file will automatically open with the **tldraw Editor**.
3. If not, right-click any .tldr file and select **Open With...** -> **tldraw Editor**.

### Iconify Lookup

Click the custom shapes button in the toolbar to toggle the Iconify search panel. Find any icon and copy it to your clipboard or use it in your drawing.
